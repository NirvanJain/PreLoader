# PreLoader

![npm version](https://img.shields.io/npm/v/preloader-nirvan)
![npm downloads](https://img.shields.io/npm/dm/preloader-nirvan)


A lightweight JavaScript website preloader.

## Installation

```bash
npm install preloader-nirvan
```

## Usage

```html
<link rel="stylesheet" href="style.css">
<script src="script.js"></script>
```

or with Node:

```javascript
const preloader = require("preloader-nirvan");

preloader();
```
